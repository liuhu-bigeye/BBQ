## 计划
地点：八零会议室暮色轰趴馆(4号馆)，昌平区北七王府花园

时间：0512(周日)自行前往14点到。晚上过不过夜均可(房间保留到次日10点)。

项目：三层别墅，一楼K歌、桌游，二楼网吧、台球、桌上足球、麻将、电视+xbox、小霸王，三楼床位休息。

饮食：烤全羊20斤 + 麻小/海底捞外卖，房间有酒水不免费

**提供三国杀、狼人杀、德国心脏病，如果有更多桌游等娱乐设施建议带来**
